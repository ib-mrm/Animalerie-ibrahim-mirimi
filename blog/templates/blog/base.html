Description des règles de fonctionnement de l'animalerie

Les animaux peuvent se trouver dans 4 états possible : affamé, repus, fatigué ou endormi. Les lieux peuvent quant à eux être remplis en vérifiant certaines contraintes d'intégrité:

- seul un animal affamé accepte d'être nourri dans le mangeoire

- seul un animal repus accepte de faire de l'exercice sur la roue

- seul un animal fatigué accepte de dormir dans le nid

- seul un animal endormi peut être réveillé. Une fois réveillé, il est placé dans la litière
